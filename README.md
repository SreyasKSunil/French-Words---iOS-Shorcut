French Words – iOS Shortcut

Overview
This iOS Shortcut helps you learn French vocabulary through random selection, audio pronunciation, and repetition control.
It uses Apple Notes as a lightweight database and avoids repeating words until all entries are used.

Features
	•	Random French word or sentence
	•	English meaning
	•	French and English example sentences
	•	Spoken pronunciation in French and English
	•	“Next” button to continue learning
	•	No repetition until all items are completed
	•	Fully offline after setup
	•	Works without Apple Intelligence

Requirements
	•	iPhone with Shortcuts app
	•	Apple Notes app
	•	iOS 18 or later
	•	iCloud Notes enabled

Files in this repository
	•	French-Words.shortcut – the actual iOS Shortcut
	•	README.md – setup and usage guide
	•	LICENSE – usage license
	•	sample_french_bank.txt – optional starter word list

Setup Instructions

Step 1. Install the Shortcut
	1.	Download the French-Words.shortcut file.
	2.	Open it using the Shortcuts app.
	3.	Tap “Add Shortcut”.
	4.	Allow permissions when prompted, especially Notes and Speech.

Step 2. Create Required Notes
Open the Notes app and create two notes in iCloud.

Note 1
Name: French Bank

Each line must follow this exact format.
FR||EN||FR_EX||EN_EX

Example.
Manquer quelque chose||To miss something||J’ai manqué le train.||I missed the train.

Note 2
Name: French Used

Leave this note empty.
It is used to track progress automatically.

Important
	•	Note names must match exactly.
	•	Notes must be stored in iCloud, not “On My iPhone”.
	•	Do not add extra blank lines.

How to Use
	1.	Run the shortcut.
	2.	Listen to the pronunciation.
	3.	Tap “Next” to see another word.
	4.	Tap “Done” to exit.

Progress Handling
	•	Each word shown is saved automatically in French Used.
	•	Words are not repeated until all entries are used.
	•	To reset progress, clear the content of French Used manually.

Customisation
You can:
	•	Add more words to French Bank at any time.
	•	Adjust speech speed inside the shortcut.
	•	Duplicate the shortcut for other languages.

Limitations
	•	Notes are not bundled with the shortcut.
	•	Each user must create their own notes.
	•	Shortcuts must be installed manually.
